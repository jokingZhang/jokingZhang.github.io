介绍

恭喜你正在经历一场冒险！

本篇AngularJS的教程意图是向你展示所有最好的Angular资源。我们的目的是使得目前的这些课程更加丰富，通过各种各样的观点，以便于给你更加全面的视角。

AngularJS的学习曲线就像一根曲棍球球棍，从应用程序的基本功能入手是很容易的。然而，构建更加复杂的应用则需要了解Angular内部的工作原理。如果不这样做，会导致开发变得笨拙，麻烦。

对于 AngularJS，那种“准备，瞄准，射击”的学习方式，比如看一些粘来粘去的教程，粗略的浏览文档。这种方式会使得学习者混乱，而且沮丧。这个课程的目的是通过高质量的内容来让读者了解Angular的方方面面。当你最终掌握了Angular 你将可以流畅，有效的构建大型应用程序。

预备知识

  • 中等程度的HTML CSS JavaScript
  • 了解基本 MVC 的概念
  • DOM(浏览器文档对象模型)
  • JavaScript 的 函数 事件 异常处理机制

资源

相对于其他的JavaScript框架，AngularJS还属于起步阶段，资源也不是特别的充足。因此，课程引入了许多精彩的博客，以便提供更生动，更耐人寻味的各自的观点。

踢轮胎

AngularJS 不是一个文库。

相反，这是一个包含更多变现力，可读格式的可扩展HTML的JavaScript框架。它允许你使用特殊标记来编写HTML，以便于自动的同步你的JavaScript应用程序逻辑，而不是手动的更新视图。如果，你想扩大现有的 JavaScript 应用，或者你想利用框架全力打造一个具有丰富交互形式的单页面应用，AngularJS 可以帮助你构建简洁有效的代码。

当开始一个全新的话题，在进入细节之前，了解其架构很重要。

阅读下面的AngularJS导读文章，他们将会告诉你如何更好的开始。不要担心他们讲到的方方面面的东西，他们将会被包含在后续的课程中。
